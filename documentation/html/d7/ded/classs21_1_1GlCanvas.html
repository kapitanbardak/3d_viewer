◆ SetProjectionType()

void s21::GlCanvas::SetProjectionType ( )

Sets the type of the projection (central/ parralel) based on current type.

Set the projection type for the OpenGL canvas.

This function sets the projection type of the OpenGL canvas based on the current settings. If the projection type is set to CENTRAL, it uses a perspective projection (glFrustum). Otherwise, it uses an orthographic projection (glOrtho).
