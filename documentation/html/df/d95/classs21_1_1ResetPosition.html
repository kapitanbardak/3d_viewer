void s21::ResetPosition::execute ( )
overridevirtual

Executes the command, resetting the model's position.

Execute the reset position command.

This method resets the position of the 3D model, initializes the GlCanvas with the updated data, and resets the values in the user interface input boxes. It then triggers a repaint of the canvas.

Implements ICommand.
